README


Проект:

    Проект написан частично на Java, частично на Scala, поэтому в качестве императивного загрузчика
    библиотек использован sbt. Соответственно все библиотеки, необходимые для работы проекта определены
    в файле build.sbt


MySQL:

    В качестве БД поставлена MySQL. Чтобы запустить в локальном варианте связь с таблицей – нужно ее
    снова создать в MySQL пользователя.

    При выполнении приложения applications.MySQL_connect - нужно выбрать добавить строку AVG в таблицу (1) или нет(0)
     (Функция добавлена в качестве примера обновления таблицы в БД)


Spark Streaming:

    Для динамической обработки данных через Spark Streaming – следует запустить какой-то источник с
    постоянно генерируемой информацией. Я реализовал это через Netcat server. Подключив к нему Spark
    Streaming , во время его работы, я набирал рандомный тест, который он обрабатывал и складывал в
    папку SparkStreaming.


Spark:

    Для анализа текста через SpakApp – нужно запустить SpakApp и прописать в нем путь к текстовому
    файлу (у меня там “text_for_spark.txt”, в котором лежит отрывок из Пруста. Результаты обработки
    текста появятся в папке SparkTextFile.


Karfka:

    Apache Karfka (Файл applications.Kafka) подсоединяется к брокерам, которые находится на локалхосте 9092
    и 9093 соответственно. Брокеры были мной реализованны также через Netcat server.

Unit тест:

    Unit тест добавлен только в качестве примера самого unit теста, сделан для отдельного от проекта приложение HelloWorld
